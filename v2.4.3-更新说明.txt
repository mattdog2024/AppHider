AppHider v2.4.3 - 更新说明
========================

更新内容
--------
1. 移除了"Emergency Disconnect"（紧急断开）快捷键设置
   - 原因：该功能已经集成到隐私模式快捷键中，不需要单独的快捷键
   - 现在只有两个快捷键：
     * Toggle Privacy Mode（切换隐私模式）- 默认 Ctrl+Alt+F9
     * Open Menu（打开菜单）- 默认 Ctrl+Alt+F10

2. 简化了UI界面
   - 移除了Emergency Disconnect快捷键的输入框和说明
   - 界面更加简洁清爽

功能说明
--------
按下隐私模式快捷键后，系统会自动执行：
1. 关闭远程桌面连接窗口（如果存在）
2. 隐藏选定的应用程序
3. 断开网络连接

所有功能都集成在一个快捷键中，使用更加方便！

保留的功能
----------
✓ 隐私模式（隐藏应用 + 关闭远程桌面窗口 + 断网）
✓ MAC 地址随机化
✓ 密码保护
✓ 开机自启动
✓ 全局快捷键（2个）
✓ 紧急恢复功能
✓ 所有安全特性

升级建议
--------
如果您正在使用 v2.4.3 之前的版本，建议升级到此版本：
- 更简洁的界面
- 更方便的操作
- 所有功能保持不变

安装方法
--------
1. 解压 AppHider-v2.4.3-SimpleRDWindowClose.zip
2. 以管理员身份运行 AppHider.exe
3. 首次运行会要求设置密码
4. 配置您的快捷键和要隐藏的应用程序

注意事项
--------
- 如果您之前配置了Emergency Disconnect快捷键，升级后该设置将不再使用
- 所有功能现在都通过"Toggle Privacy Mode"快捷键触发
- 紧急恢复按钮仍然保留在主界面上，可以随时使用
